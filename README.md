# OVD-Study

Open-Vocabulary Detection(OVD) 모델을 공부하고 정리하는 레포
Inspired by https://arxiv.org/pdf/2401.17270

## Scope
- YOLO-World, GLIP, Grounding DINO, Detic 등 OVD/VLM 기반 디텍터
- region–text pretraining, prompt-then-detect, zero-shot transfer 분석

## 나의 의견
- Zero-Shot이 가능한 Object Detector
- 단순 CNN의 한계를 뛰어 넌는 모델로 판단됨(개인생각)
